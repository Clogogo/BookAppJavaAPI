# E-Travel Web Application 
Backend configuration REST API 
@Rest controller 
* Authentication
* User Management


## Requirements

For development, you will only need installed the following on your environment.
* Java 11
* Spring Boot
* lombok
 

## Run backend
```
$ maven clean
$ maven build
```
